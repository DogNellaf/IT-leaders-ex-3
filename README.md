# Классификация Fashion MNIST с помощью CNN (Keras)

Учебный проект по теме сверточных нейронных сетей: построение классификатора изображений одежды на датасете Fashion MNIST.

## О проекте

В ноутбуке реализован полный базовый ML-пайплайн:

- загрузка данных Fashion MNIST;
- визуализация примера изображения;
- предобработка (нормализация, изменение формы тензоров, one-hot кодирование меток);
- построение и компиляция CNN-модели на Keras;
- обучение модели;
- оценка качества на тестовой выборке (accuracy, precision, recall, F1-score).

## Структура

- `practice.ipynb` - основной ноутбук с решением.

## Используемые библиотеки

- numpy
- matplotlib
- tensorflow / keras
- scikit-learn

## Запуск

1. Установите Python 3.9+.
2. Установите зависимости:

   ```bash
   pip install numpy matplotlib tensorflow scikit-learn jupyter
   ```

3. Запустите Jupyter:

   ```bash
   jupyter notebook
   ```

4. Откройте `practice.ipynb` и выполните ячейки по порядку.

## Архитектура модели (базовый вариант)

- Conv2D: 32 фильтра, ядро (4, 4), ReLU
- MaxPooling2D: pool_size (2, 2)
- Flatten
- Dense: 128, ReLU
- Dense: 10, Softmax

Параметры компиляции:

- loss: categorical_crossentropy
- optimizer: RMSprop
- metrics: accuracy

## Что можно улучшить

- добавить валидационную выборку и раннюю остановку (EarlyStopping);
- поэкспериментировать с глубиной сети, числом фильтров, Dropout и BatchNormalization;
- построить матрицу ошибок и визуализацию ошибок классификации;
- выполнить дополнительное задание: визуализировать фильтры Conv2D до и после обучения.

## Цель работы

Закрепить практику работы с CNN в Keras и научиться оценивать качество многоклассовой классификации изображений.
